Detailed Instructions



Uploading Book

1. Enter book name and author.
2. Choose the language of the book to be uploaded.
3. Click on "Choose File" and select the book to be uploaded by navigating in the file system.Kindly note that only PDF input is supported.

Recording audio

1. Select the book to record
2. Allow the browser to use the microphone in the popup at the top.
3. Recording a chunk can be done in 2 ways:
  a) You can use the recorder in the browser: Selecting "Record Audio" will start the recording. Select "Stop Recording" to stop the recording. Then choose "Save audio". This will save the audio file in your Downloads (usually) and you can then choose the file from there while uploading.
  b) You can also make use of the recorder on your machine: Now you can directly select the audio file you want to upload. 4. Choose if the chunk is the beginning of chapter or not
5. Upload the chunk

Digitizing Book

1. Select the book to digitize
2. Type the text shown in image, in the text box provided.For help, click on the letter adjacent to the language drop down.Choose either Quick Help or Detailed Help.
3. Choose if the chunk is the beginning of chapter or not
4. Upload the chunk

Validating Book

1. Select the book to validate as per your choice
2. Select a paragraph from the pool of parts
3. Listen to the audio against the image shown .Upvote/Downvote by clicking Good/Bad depending on the clarity and the accuracy of the audio clip

Browsing Book

1. Select either Audio or Digitized books.
2. Select the book you want to view or download.
3. The audio book/digitized book is arranged in the order of chapters
4. Click on the desired chapter to either view or download the same
